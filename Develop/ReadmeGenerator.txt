
      # ReadmeGenerator
      
      ## Table of Contents
      
      ### A simple application to create a basic readme file that can be uploaded to your projects github portfolio including title, description, installation notes, testing notes, any contribution details and a method of contact all rolled into a .md file
      
      ### Copy this repo into your project and simply run 'node index.js' in the command line or terminal and follow the prompts
      
      ### None
      
      ### Follow the guide on how to install and run the application but omit any sensitive or personal information for the initial test
